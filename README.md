# Java_Base64
JavaでBase64を扱う

## エンコード

```
mvn clean compile exec:java -Dexec.mainClass="sample.java.base64.Base64Encoder"
```

## デコード

```
mvn clean compile exec:java -Dexec.mainClass="sample.java.base64.Base64Decoder"
```
